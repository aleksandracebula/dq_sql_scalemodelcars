# 🏎️ Scale Model Cars Database Analysis
### SQL Guided Project | Dataquest

## 📌 Project Overview
The purpose of this project is to showcase my SQL skills. It was developed as part of a data science guided course on the Dataquest platform. I have analyzed data from a sales records database for scale model cars. 

### Key Objectives:
1. Inventory Management: identifying which products should have been ordered more or less frequently based on sales performance.

2. Marketing Strategy: tailoring communication and marketing strategies to specific customer behaviors and segments.

3. Customer Acquisition Cost (CAC): determining the optimal budget for acquiring new customers by analyzing current lifetime value.

---

## 🗄️ Database Schema
The database consists of 8 tables:

* `Customers`: Profile data of clients.
* `Employees`: Corporate hierarchy and staff info.
* `Offices`: Global sales office locations.
* `Orders`: Transaction headers.
* `OrderDetails`: Line items for each specific order.
* `Payments`: Financial transaction history.
* `Products`: Catalog of scale model cars.
* `ProductLines`: Category-level descriptions.

---

## 🛠️ Analysis Steps

### Step 1: Data exploration
Understanding the scale and structure of the data.
* Verified table dimensions (number of attributes and rows).
* Analyzed relationships between customers, orders, and products.

### Step 2: Exploring products
**Goal:** Optimize the supply chain by identifying "understocked" but "high-performing" items.
* **Low Stock Ratio:** Calculated by dividing the sum of quantities ordered by the quantity in stock.
* **Product Performance:** Ranked products by total sales revenue.
* **Restocking Priority:** Filtered products with high performance and low stock ratios to prevent revenue loss.

### Step 3: Customer Segmentation
**Goal:** Designing a Customer Reward Program (CRP) based on engagement and profitability.
* Analyzed **net profit** generated by each customer.
* Identified the **Top 5 Customers** for VIP rewards.
* Identified the **Top 5 Least-Engaged Customers** to develop re-engagement marketing campaigns.

### Step 4: Financial Metrics (LTV & CAC)
How much can be spent on acquiring new customers?
The idea was to find:
* a number of new customers arriving each month,
* finding Customer Lifetime Value (LTV) which tells how much profit an average customer generates during their lifetime with the store.
